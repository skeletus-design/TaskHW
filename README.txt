Данный мод актуален для версии Minecraft 1.16.1 и требует установку Forge версии 32.0.108

Для большей наглядности я также загрузил в отдельный репозиторий файлы для разработки https://github.com/skeletus-design/firstmod

Говоря о возможностях мода: 
Мод добавляет новый предмет - Цитрин, который не имеет на данной версии каких либо применений, однако для модификации создана отдельная вкладка в креативе и добавлена локализация на русском и английском(US) языках
